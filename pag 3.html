Monitor de frecuencia cardíaca

Es un reloj, por lo que puede verlo mientras mantiene la vista en la carretera, y es inalámbrico, por lo que no se distorsionan las lecturas, realmente funciona para correr y le permite realizar un seguimiento a toda velocidad.
La pequeña placa de desarrollo Arduino se ha combinado junto con diversos componentes desde diversos leds hasta un módulo nRF24 genérico junto con un anillo NeoPixel Ring de Adafruit, lo que ayuda al usuario, ya que le proporciona una retroalimentación visual inmediata de tu frecuencia cardíacae incluso cuenta con un grado de luz, es decir, rojo significa presión elevada y azul, presión normal.

chaqueta indicadora para ciclistas

Esta Chaqueta de Ciclismo Interactiva también viene con LEDs delanteros y traseros para mantener visible al ciclista en todo momento,además de contar con iluminación y conexión a internet también repele el agua, es un producto completo, pero cuenta con un inconveniente ya que depende del uso de datos del celular para tener todas sus funciones permanentes, no hablando solamente del acceso a internet si no la complejidad de utilizarlo mientras se conduce. Para realizar este proyecto vas a necesitar una placa Arduino UNO o LilyPad, dos acelerómetros, luces LED, y algunos otros componentes para completar este proyecto.

Alarma para casa

Sistema de alarma basado en Arduino cuenta con un sensor de movimiento, Un arduino, un sensor ultrasónico de "ping", estoy usando HC-SR04 Un PIR sería mejor, pero esos son caros. Un sensor de ping se puede colocar subrepticiamente en una puerta y aún cumplir el mismo trabajo básico, un zumbador piezoeléctrico, una tira de luz LED, con el mismo cableado que usamos en este proyecto
Esta alarma puede detectar cualquier atisbo de movimiento, en caso de que se detecte algo sonará un sonido de alarma muy agudo proviniente del zumbador y se encenderan las diferentes luces intermitentes.

BASTON INTELIGENTE

Detectar obstáculos. Viene equipado por varios sensores y un motor de vibración que advierte al usuario de la proximidad de un obstáculo.
Se conecta directamente a un teléfono móvil inteligente a través de su conectividad Bluetooth. Cuenta con varios tipos de sensores de proximidad, giroscopio y acelerómetro para «leer» el entorno. Reconoce obstáculos en una distancia de hasta 250 centímetros, incorpora, además, una pequeña pantalla desde donde se pueden consultar las indicaciones de la ruta escogida a través del servicio digital Google Maps.
