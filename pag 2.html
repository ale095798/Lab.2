En esta clase veremos algunos principios de la programación, aprenderemos
las partes de HTML y CSS, introducir imágenes, videos y cambiar información

HTML (HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

!DOCTYPE

Indica al navegador que el documento está basado en el estándar HTML5

head

Se usan para definir la cabecera la página web

title

Se utiliza para definir un título para el documento

body

Define la parte corporal o cuerpo de un documento

imágenes

Etiqueta para vincular una imagen en la página web

vínculos

Se usa para enlazar recursos externos al documento HTML
